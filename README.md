# React + Tailwind + JWT Protected
- Login -> POST /auth/token/
- Auto refresh -> POST /auth/token/refresh/
- Protected routes with axios interceptor
- Demo protected call: GET /api/me/

## Run
npm i
npm install react-icons
cp .env.example .env
npm run dev
